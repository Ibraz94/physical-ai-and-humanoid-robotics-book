# Introduction to Physical AI

**Learning Objectives**

## What is Physical AI?

Physical AI refers to systems where artificial intelligence algorithms interact directly with the physical world through sensors and actuators.

```python
# Physical AI Concept: Sensor-Action Loop
class Robot:
    def __init__(self):
        self.state = "idle"

    def sense(self, sensor_data):
        # AI processing of physical data
        return self.process(sensor_data)

    def act(self, command):
        # Physical actuation
        print(f"Executing: {command}")
```

## Why ROS 2?

ROS 2 (Robot Operating System 2) is the industry standard middleware for robotics. We will use the **Jazzy Jalisco** distribution.

```bash
# Check your ROS 2 installation
printenv | grep ROS
# Expected output: ROS_DISTRO=jazzy
```

## Your First Node

Here is a simple Python node in ROS 2:

```python
import rclpy
from rclpy.node import Node

class MinimalNode(Node):
    def __init__(self):
        super().__init__('minimal_node')
        self.get_logger().info('Physical AI Course Node Started!')

def main(args=None):
    rclpy.init(args=args)
    node = MinimalNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```
