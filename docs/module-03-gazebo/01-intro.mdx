# Module 3: Open-Source Simulation with Gazebo

**Learning Objectives**
- Understanding Gazebo (formerly Ignition) architecture.
- Launching a simulation world.
- Spawning a robot entity.

```bash
# Launching Gazebo
ign gazebo empty.sdf
```

```xml
<!-- Simple SDF World -->
<sdf version="1.6">
  <world name="default">
    <include>
      <uri>https://fuel.ignitionrobotics.org/1.0/OpenRobotics/models/Sun</uri>
    </include>
  </world>
</sdf>
```

```python
# Spawning via Python (ros_gz_interfaces)
import subprocess

def spawn_entity():
    subprocess.run(["ros2", "run", "ros_gz_sim", "create", "-topic", "robot_description", "-entity", "my_robot"])
```
